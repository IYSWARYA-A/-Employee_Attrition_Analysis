#Employee_Attrition_Analysis

## Project Overview

## Project Title : Employee_Attrition_Analysis

### Objectives
- Analyze employee attrition trends in the organization
- Identify key factors affecting employee turnover
- Understand workforce distribution across departments and roles
- Support HR decision-making using data-driven insights

### Tools & Technologies Used
- Python – Data cleaning and preprocessing (Pandas, NumPy)
- SQL – Data querying and analysis
- Power BI – Dashboard creation and data visualization

### Objectives of the Project
- Clean and preprocess raw employee data
- Perform exploratory data analysis (EDA)
- Analyze attrition based on:
Department
Job Role
Salary
Experience
Overtime
Job Satisfaction
Build an interactive dashboard for HR insights

## Features
- Data cleaning and transformation using Python
- SQL queries for deep analysis (aggregation, filtering, grouping)
- Interactive Power BI dashboard with slicers

## Output
- Cleaned dataset ready for analysis
- SQL-based insights on employee trends
- Interactive Power BI dashboard showing:
- Workforce distribution
- Attrition patterns
- Job satisfaction levels
- Promotion and retrenchment analysis
- Easy-to-understand visuals for decision-making
